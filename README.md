# Library Management System - RESTful API

A comprehensive RESTful API for managing genres, authors, and books with JWT authentication and role-based access control.

## Features

- **Complete CRUD Operations** for genres, authors, and books
- **JWT-based Authentication** with role-based access control
- **API Versioning** with v1 endpoints
- **Swagger/OpenAPI Documentation** for easy API exploration
- **Docker Containerization** for easy deployment
- **PostgreSQL Database** with persistent storage
- **Comprehensive Testing** with unit tests
- **Multi-environment Configuration** (dev, test, prod)

## Technologies Used

- Java 17
- Spring Boot 3.1
- Spring Security
- Spring Data JPA
- PostgreSQL
- Docker & Docker Compose
- JWT for authentication
- Swagger/OpenAPI for documentation
- JUnit 5 for testing

## Prerequisites

- Java 17+
- Maven 3.8+
- Docker & Docker Compose
- PostgreSQL (if running without Docker)
